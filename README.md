## ✨ 特長

- 左に追従する目次（現在地ハイライト）＋上部に読了プログレスバー
- コピーできるプロンプト例を多数収録
- チェックボックス付き練習リスト、開閉式の用語集・FAQ
- 外部依存なしの**自己完結HTML**（`index.html` を開くだけで動作）

## 🚀 見る
